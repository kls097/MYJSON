**生成日期**: 2026-01-30
**最后更新**: 2026-02-10
**当前版本**: 4.0.0
**报告类型**: 竞品分析与功能优化建议

---

## 一、执行摘要

本报告通过对市场主流 JSON 工具的调研分析，结合当前项目已有功能，提出针对性的功能优化和新增建议。目标是提升用户体验，增强产品竞争力，使本插件成为 uTools 平台上最全面的 JSON 处理工具。

---

## 二、当前功能盘点

### 2.1 核心功能（已实现）

#### ✅ 基础编辑功能

- JSON 格式化（美化）
- JSON 压缩（Minify）
- 实时语法验证
- 语法错误提示（行/列定位）
- 撤销/重做（CodeMirror 原生支持）
- 键名排序（递归排序嵌套对象，保持数组顺序）

#### ✅ 高级编辑功能

- JSON5 注释移除
- 转义字符处理（智能反转义）
- 压缩并转义（一键压缩+引号转义）
- 6级渐进式 JSON 修复
  - Level 1: 标准 JSON.parse
  - Level 2: jsonrepair 库修复
  - Level 3: JSON5 解析
  - Level 4-6: 深度预处理组合

#### ✅ 查询功能

- JSONPath 查询（jsonpath-plus）
- JMESPath 查询（jmespath）
- 查询结果面板展示
- 路径提取功能

#### ✅ 可视化功能

- 代码编辑器视图（CodeMirror 6）
- 树形视图（可折叠/展开）
- 表格视图（针对 JSON 数组）

#### ✅ JSON Schema 工作台

- Schema 生成（从 JSON 样本自动推断）
  - 多版本支持：Draft-04、Draft-06、Draft-07、2019-09、2020-12
  - 必填字段策略：全部必填/智能推断/全部可选
  - 可选添加示例值和描述
  - 多样本 Schema 合并（`mergeSchemas`）
  - `$ref` 引用优化处理
- Schema 验证（基于 Ajv）
  - 严格模式切换
  - 错误路径定位和详细消息
  - 支持 ajv-formats 扩展（email、uri、date-time 等）
- Mock 数据生成（基于 json-schema-faker）
  - 批量生成（1/5/10/50/100 条）
  - 多语言区域（中文/英文/日文/韩文）
  - 输出格式：单对象/数组
  - 预览模式
- Schema 模板库
  - 8 个内置模板（用户、订单、商品、API响应、分页、文件上传、评论等）
  - 自定义模板保存/加载（uTools DB 或 localStorage）

#### ✅ 数据转换

- JSON 转多种编程语言类型定义（14种语言）
  - TypeScript, Java, Go, Swift, Kotlin, Dart, Rust
  - C#, C++, Flow, Python, Ruby, Scala3, Objective-C
  - 使用 quicktype-core 引擎
- JSON ↔ Excel 双向转换
  - 支持 .xlsx 格式
  - 数组数据与表格互转

#### ✅ 比较功能

- JSON 结构差异对比（动态规划对齐算法）
- 并排视图展示
- 差异高亮显示
- 差异导航（上/下跳转）
- 双侧键名排序
- 相似度评分

#### ✅ 数据管理

- 历史记录保存（uTools 数据库）
- 云同步支持
- 导入/导出本地文件（JSON/TXT）
- 剪贴板操作
- 保存到本地文件（Ctrl+S 快捷键）

#### ✅ 快捷键支持

- `Ctrl+S` / `Cmd+S`：保存到历史
- `Ctrl+Z` / `Ctrl+Y`：撤销/重做（CodeMirror 原生）
- `Ctrl+Shift+G`：生成 Schema（Schema 面板内）
- `Ctrl+Shift+V`：验证数据（Schema 面板内）
- `Ctrl+Shift+M`：生成 Mock（Schema 面板内）
- `Enter`：执行查询（查询面板内）
- `↑` / `↓`：搜索结果导航（树形视图内）
